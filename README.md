SmartBite AI: Food Waste Forecasting 📉
SmartBite AI is a predictive prototype designed to reduce hotel food waste by forecasting daily production requirements. Using Google Gemini, the system analyzes historical trends and day types (Normal, Weekend, or Festival) to provide accurate cooking quantities.

Core Features
Predictive Analytics: Forecasts food demand to prevent overproduction.

Contextual Logic: Adjusts estimates based on holidays and weekend trends.

Modern UI: Built with a responsive Glassmorphism interface.

Cloud Hosted: Deployed via Firebase for global access.

Tech Stack
AI: Gemini 1.5 Flash API

Frontend: HTML, CSS, JavaScript

Hosting: Firebase
